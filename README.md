# roast-reviews

[![Build Status](https://travis-ci.org/dfmcphee/roast-reviews.svg?branch=master)](https://travis-ci.org/dfmcphee/roast-reviews)

Coffee bean reviews.

## Dependencies

* Ruby 2.3.1
* MySQL

## Getting started

`bin/rake db:setup`

`bin/rails server`

## Testing

`bin/rails db:test:prepare`

`bin/rails test`
